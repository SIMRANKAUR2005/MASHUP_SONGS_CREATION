# MASHUP_SONGS_CREATION
This project implements a YouTube mashup generator]
It downloads videos of a given singer from YouTube, extracts audio, trims clips, and merges them into a single mashup file.

The project contains two components:

-Program 1 — Command-line Python mashup generator
-Program 2 — Web service mashup generator

**Features**
-Download multiple YouTube videos automatically
-Convert videos to audio
-Trim audio clips to fixed duration
-Merge audio files into one mashup
-Input validation and exception handling
-Simple web interface for mashup generation
-ZIP file delivery via email (Program 2)

**Technologies Used**
Python
yt-dlp
MoviePy
Pydub
FFmpeg
Flask
